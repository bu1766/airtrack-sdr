# Contributing

Contributions are welcome under GPL-2.0-or-later.

Before opening a pull request:

1. Keep all user-facing text in English.
2. Run `./scripts/check-english.sh`.
3. Run `./scripts/build-macos.sh` and `./scripts/verify-bundle.sh` on macOS.
4. Test Start, Stop, reconnect, no-device, and device-busy states.
5. Do not claim hardware support without a working decoder backend and a documented validation path.
