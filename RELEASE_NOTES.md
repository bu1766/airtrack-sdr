# AirTrack SDR 1.1.0

## 简体中文

AirTrack SDR 1.1 为简体中文用户增加了内置本地化，同时继续保持英文为默认界面。

### 本次更新

- 在设备控制区新增语言选择器：English / 简体中文
- 中文化设备连接、状态提示、航班路线、机型、飞机详情、搜索和常用地图设置
- 自动保存语言选择，下次启动继续使用
- GitHub 首页改为中文优先、英文对照，并补充中文安装与故障排除说明
- 保持本地解码、设备自动发现和多设备选择等 1.0 功能不变

## English

AirTrack SDR 1.1 adds built-in Simplified Chinese localization while keeping English as the default interface.

### Changes

- Added an English / 简体中文 language selector to the device control panel
- Localized receiver controls, status messages, route and aircraft details, search, and common map settings
- Saves the selected language locally for the next launch
- Reworked the GitHub landing page with Chinese-first documentation and an English reference section
- Preserved local decoding, automatic device discovery, and multi-device selection from version 1.0

## Installation note

Version 1.1.0 community binaries are ad-hoc signed but not Apple-notarized. On first launch, macOS may require right-clicking **AirTrack SDR** and choosing **Open**. Apple Developer Program membership is not required to use the app.
